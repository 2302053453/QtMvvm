Detailed Description

An extension of the Label for better appearance when used in a toolbar.

Since
1.0

This special label is bigger in size and correctly aligned for use in a toolbar. You can either let it fill the whole toolbar or use it in a layout with Layout.fillWidth: true and it will perfectly fill up the bar like a normal toolbar label for mobile activity titles.
