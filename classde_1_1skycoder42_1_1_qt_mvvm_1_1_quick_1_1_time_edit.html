Detailed Description

A edit view to edit times by using tumblers.

Since
1.1

You can use this edit if you want to let the user select a time. You can use the properties to control wich tumblers are shown and which format they use

◆ use24HourFormat

bool de::skycoder42::QtMvvm::Quick::TimeEdit::use24HourFormat

Holds the time format to use to edit the time.

Default: locale dependend

In 24h format, the hour tumbler shows 0-24 as numbers. In the 12h format, it shows the numbers 1-12 and an additional tumbler with "am" and "pm" is shown to the far right. The correct version is detected from the locale, but you can always overwrite it to fit your needs.
