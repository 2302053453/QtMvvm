The view implementation for the QtMvvm::SettingsViewModel.

This is the view used to present a settings view model. You can extend the class if you need to extend that view.

bool de::skycoder42::QtMvvm::Quick::SettingsView::closeAction ( )

Can be called to try to close a single settings page.

Returns
true if a page was closed, false if not

This method is called by the presenter to close the pages of the settings view one at a time.

Note
if SettingsView::fullClose is true, this method will always return false.

◆ fullClose

bool de::skycoder42::QtMvvm::Quick::SettingsView::fullClose

Specifiy if a back action should always close the full settings.

Default: false

The settings view consists of an internal stack view to present settings pages. By default only one page is closed at a time. By setting this property to true, a close action will always close all of them.
