Detailed Description

An extension of the ActionButton to provide a simple "more menu" button.

Since
1.0

The button already has a text, icon and tooltip and it will automatically display a context menu when pressed. Add the menu items as the children of the button:

MenuButton {
id: mButton
MenuItem {
text: "Copy"
}
MenuItem {
text: "Paste"
}
//...
}
