Detailed Description

A QML class access the native file chooser on android.

Since
1.0
Warning
Available on android only!

A C++ wrapper around the android java API to access the native file chooser. It can only be used on Android, and serves as the base for the android implementation of the file and folder dialogs.

See also
FileDialog, FolderDialog
