Detailed Description

A Drawer that can be used as a presenter for drawer views.

Since
1.0

You can use this class as presenter for drawer views. Those are views that are not drawers themselves, but need to be placed in one in order to properly work. Use it as follows:

MyPresenter {
id: _root
PresentingDrawer {
id: _rootDrawer
}
function presentDrawerContent(item) {
return _rootDrawer.presentDrawerContent(item);
}
function toggleDrawer() {
_rootDrawer.toggle();
}
function closeAction() {
var closed = false;
// ...
if(!closed)
closed = _rootDrawer.closeAction();
// ...
return closed;
}
}
